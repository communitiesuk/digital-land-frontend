Digital land frontend

Get started

The Digital land frontend is collection of reusable frontend parts and examples created for the digital land team.

It builds on the GOV.UK Design System and should be simple to use if you are familiar with that.

It has 3 main components:

  • Jinja components - the GOV.UK Design System components as jinja macros.
  • Components - components specific to Digital Land.
  • Javascript - scripts written to use with Digital Land prototypes and services.
