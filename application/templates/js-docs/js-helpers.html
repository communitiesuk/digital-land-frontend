Helpers

A few options for setting up JS components.

Using jQuery

Digital land frontend has a version available. To include it in a template set the includeJQuery jinja variable to 'true'.


        {%- filter forceescape -%}{%- raw -%}
        {% set includeJQuery = "true" %}
        {%- endraw -%}{%- endfilter -%}

This will include version 3.4.1.

Google analytics

Google analytics for Digital Land is included by default. If you want to change the GA setup or exclude you can do so by changing the {%- raw -%}{% block googleAnalytics %}{% endblock %}{%- endraw -%} jinja block (see dlf-base template).
