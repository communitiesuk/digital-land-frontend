Back to top

A component to help the user get back to the top of a long page.

Back to top banner example

Here is an example of a long page with the back to top component a banner that is visible to the user as they scroll down the page. It works by defining a head and footer element. When the user scolls between these elements the banner will be displayed.

JS 
<script>
      var $btt = document.querySelector('[data-module*="back-to-top"]')
      new window.DLFrontend.BackToTop($btt).init({
        footer_selector: ".govuk-footer",
        head_selector: ".test-hub",
        classes: "back-to-top--banner"
      });
    </script>
