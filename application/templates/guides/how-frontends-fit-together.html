Which frontend assets do I need to use?

Digital land frontend includes compiled assets and the building blocks that produce the assets.

Knowing frontend requirements of your project will help determine what you need to include.

Broadly, there are 2 types of projects

  1. you only need to use the digital land components
  2. you need custom css and to build custom components

I only want to use the digital land components

Including all the compiled assets in your project will allow you to use all the digital land components as well as the GOVUK Design System components.

To get the most from what is available, include

  • dl-frontend.css — css for styling pages
  • dl-frontend.js — available JS modules for progressively enhancing components
  • /dl-macros — jinja macros to help produce the right HTML for components
  • /dl-partials — HTML partials to include in your templates
  • govuk-jinja-components — python module for including all the GOVUK design sytem components as jinja components

I want to build custom components

Instead of including the compiled assets (dl-frontend.css, dl-frontend.js) you will need to include the precompiled scss and js modules.

Depending on your build process you might find gulpfile.js useful. It is a task runner that will help you build assets.
