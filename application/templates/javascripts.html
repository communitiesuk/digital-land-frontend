Javascripts

There are a few javascript components used with Digital land projects.

  • DL Frontend

    A set of modules to progressively enhance components.

  • Back to top

    Progressively enhances back to top link, useful for long pages.

  • MHCLG Maps

    A wrapper for leafletJS that makes it quicker to render a map.

  • Accessible Autocomplete

    Turns select elements into autocomplete fields.
