Standard size search box

This is the standard searchbox component. It is a maximum of 450px wide. See below if you'd like search boxes with different widths.

Please note To make this component accessible you will need to handle invalid entries, setting aria-invalid to true on the input when invalid.

Alternative sizes

You can create your own search box sizes by using the @search-box mixin. E.g.

  .search-box--small {
    @include search-box($max-with: 300px);
  }

This will result in a smaller search box used via a .search-box--small class.
